# 📊 NFHS-5 District Health Data Analysis 🇮🇳

This project analyzes India’s **NFHS-5 (National Family Health Survey-5)** district health dataset using Python to understand how **women’s education influences major health indicators** across different districts of India.

It focuses on identifying meaningful patterns between education and public health outcomes using **data visualization, statistical analysis, and linear regression models**.

---

## 🎯 Project Objectives

The main goal of this project is to study how improving women’s education can create a positive impact on health and social development.

### Key Indicators Analyzed:
 
* 👧 Child Marriage
* 🧒 Child Stunting
* 🩸 Anaemia Levels
* 🤱 Maternal Health
* 👶 Child Health Indicators
* 🎓 Women’s Education Levels

## 📈 Model Used for Prediction

This project mainly uses **Linear Regression** to predict and understand the relationship between women’s education and important health indicators like child marriage and child stunting.

### Linear Regression Equation:

**y = mx + c**

Where:

* **y** = Dependent Variable (Health Indicator)
* **x** = Independent Variable (Women’s Education)
* **m** = Slope
* **c** = Intercept

--- 

## 🌍 Project Outcome

The analysis shows that higher women’s education is strongly linked with better health outcomes, reduced child marriage, and improved maternal and child health.

This project highlights how **education can be one of the strongest tools for social and health transformation in India.**
